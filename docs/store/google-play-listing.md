# ZayCho Google Play Listing Draft

## App name

ZayCho

## Short description

Myanmar grocery storefront with wishlist, cart, and assistant support.

## Full description

ZayCho is a mobile-friendly Myanmar grocery storefront designed for fast browsing and simple product discovery.

Use ZayCho to:

- explore featured products
- search across categories
- save wishlist items
- prepare a shopping cart
- get help from the built-in assistant

The experience is optimized for quick shopping sessions and clean navigation on phone screens.
