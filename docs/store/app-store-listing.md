## Promotional text

## Description

ZayCho helps customers discover Myanmar grocery products in a fast, mobile-friendly shopping experience.

What you can do:

- browse featured grocery items
- search products by name or category
- save favorites to a wishlist
- prepare a shopping cart before checkout
- ask the in-app assistant for help finding products

ZayCho is designed for a lightweight storefront experience with a clean interface that feels natural on both phones and tablets.
